generate_keypair ([key_type[, opts]])
Generate new identity keypair. opts.bits (number) configures RSA key size (default 2048). opts.curve (string) configures ECDSA curve (default prime256v1).

Parameters:

  • key_type string ed25519|rsa|ecdsa|secp256k1. (optional)
  • opts table (optional)

Returns:

  1. table or nil keypair
  2. table err
