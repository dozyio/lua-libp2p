Client:_exchange (op, payload, lifetime, opts)
Internal PCP request/response exchange. opts.gateway, opts.port, opts.timeout, opts.retries, and opts.source_ip override client defaults.

  • payload

Client:map_port (protocol, internal_port[, suggested_external_port[, lifetime[, suggested_external_ip[, opts]]]])
Request PCP MAP mapping. opts.gateway overrides configured gateway. opts.source_ip pins local source address for UDP socket binding.

Parameters:

  • protocol string tcp|udp.
  • internal_port number
  • suggested_external_port number (optional)
  • lifetime number (optional)
  • suggested_external_ip string (optional)
  • opts table (optional)

Returns:

  1. table or nil mapping
  2. table err
