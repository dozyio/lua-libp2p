Module lua_libp2p.multiaddr

Multiaddr parsing, formatting, and helpers.

Functions

decapsulate (base, suffix) Remove a trailing suffix multiaddr.
encapsulate (base, extra) Encapsulate one multiaddr inside another.
format (addr) Format a parsed multiaddr back to text.
from_bytes (bytes) Decode binary multiaddr bytes to text and components.
is_private_addr (input) Return true when address is private/loopback/link-local.
is_public_addr (input) Return true when address is considered public-routable.
is_relay_addr (input) Check whether an address is a relay (/p2p-circuit) address.
parse (text) Parse a multiaddr string into components.
relay_destination_addr (relay_addr, destination_peer_id) Build a relay destination address for a target peer.
relay_info (input) Extract relay components from a relay destination address.
relay_reservation_addr (relay_addr) Build a relay reservation address from a relay peer address.
