Functions

Client:_discover (opts)
Internal AutoNAT discovery/check execution. opts.max_autonat_servers, opts.target_autonat_responses, and opts.stop_on_first_reachable shape check strategy. opts.discovery_timeout_seconds, opts.check_timeout, opts.drain_seconds control timing.

Parameters:

  • opts
Client:_discover_servers (limit, opts)
Select candidate AutoNAT servers from peerstore/discovery. opts.checked_peers tracks cooldown state by peer id. opts.checked_peer_cooldown_seconds suppresses recently checked peers. opts.required_addr_proto filters by ip4/ip6 preference.

Parameters:

  • limit
  • opts
Client:check (server[, opts])
Run a synchronous AutoNAT check against one server. opts may include stream_opts, timeout, poll_interval, nonce, addrs, allow_private_addrs, and allow_relay_addrs.

Parameters:

  • server string or table Server peer target.
  • opts table Check options. (optional)

Returns:

  1. table or nil result
  2. table err
Client:start ()
Start AutoNAT service handlers.

Returns:

  1. true or nil ok
  2. table err
Client:start_check (server[, opts])
Spawn an asynchronous AutoNAT check task.

Parameters:

  • server string or table Server peer target.
  • opts table Same options as check. opts.stream_opts.ctx is injected automatically when omitted. (optional)

Returns:

  1. table or nil task
  2. table err
Client:start_discovery ([opts])
Run AutoNAT server discovery and checks. opts may include discovery limits/timeouts and check strategy knobs such as max_autonat_servers, target_autonat_responses, check_timeout, stop_on_first_reachable, and drain_seconds.

Parameters:

  • opts table Discovery options. (optional)

Returns:

  1. table or nil task
  2. table err
Client:start_monitor ([opts])
Start ongoing AutoNAT monitoring. opts may include monitor cadence: retry_interval_seconds, healthy_interval_seconds, min_success_rounds, and per-round discovery/check options used by start_discovery.

Parameters:

  • opts table Monitor options. (optional)

Returns:

  1. table or nil task
  2. table err
Client:stop ()
Stop AutoNAT service activity.

Returns:

    true
new (host[, opts])
Construct a new AutoNAT client/service instance. opts may include protocol limits and behavior flags such as max_message_size, allow_dial_data, max_dial_data_bytes, dial_data_chunk_size, and monitor_on_start. Additional monitor options: monitor_start_opts / monitor_opts.

Parameters:

  • host table Host instance.
  • opts table (optional)

Returns:

  1. table or nil client
  2. table err
