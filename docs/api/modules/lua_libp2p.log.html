configure (spec)
Configure subsystem logging. spec accepts debug, info, warn, error, comma-separated subsystem names, or subsystem=level pairs. */all sets the fallback level. Examples: kad_dht,host, kad_dht=debug,host=info,*=warn.

Parameters:

  • spec string
